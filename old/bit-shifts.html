Bit Operations

These are the most common operations for interacting with data on the bit level. All of the opcodes for these instructions take up two bytes. Assume “any 8-bit register” to be one of A, B, C, D, E, H or L


 
 RL R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate left
Description
Rotates the contents of r left by one bit. The leaving (7th) bit is moved into the carry and the carry is moved into the first bit of r.
Uses
Reading bit-wise data, most significant bit first		 

 
 RR R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate right
Description
Rotates the contents of r right by one bit. The leaving (0th) bit is moved into the carry and the carry is moved into the last bit of r.
Uses
Reading bit-wise data, least significant bit first		 

 
 RLC R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate left with carry
Description
Rotates the contents of r left by one bit. The leaving (7th) bit is copied into the carry, and into the last bit of r.
Uses
Reading bit-wise data, most significant bit first, while preserving the original data (after it is performed eight times)		 

 
 RRC R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate right with carry
Description
Rotates the contents of r right by one bit. The leaving (0th) bit is copied into the carry, and into the last bit of r.
Uses
Reading bit-wise data, least significant bit first, while preserving the original data (after it is performed eight times)		 

 
 SLA R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Shift left arithmetic
Description
Rotates the contents of r left by one bit. The leaving (7th) bit is moved into the carry, and the first bit of r is reset.
Uses
Very quickly multiplying an 8-bit number by two. If the number becomes greater than 255, the carry flag will be set (else reset).		 

 
 SRA R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate left
Description
Rotates the contents of r right by one bit, but the 7th bit stays the same. The leaving (0th) bit is moved into the carry.
Uses
Maybe helpful with signed integers? Potentially useful to set all bits of A to equal the 7th bit?		 

 
 SRL R
Arguments
r - Any 8-bit register or (HL),(IX),(IY),(IX+n)
Summary
Rotate left
Description
Rotates the contents of r right by one bit. The leaving (0th) bit is moved into the carry, and the first bit of r is reset.
Uses
Very quickly dividing an 8-bit number by two. If the number was odd, the carry flag will be set (else reset).
