 RRC R
Arguments r - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Rotate right with carry
Description Rotates the contents of r right by one bit. The leaving (0th) bit is copied into the carry, and into the first bit of r.
Uses Reading bit-wise data, least significant bit first (right-to-left), while preserving the original data (after it is performed eight times)
Byte Count 2
Clock Cycles
Operand RRC RRC.L RRC.S
(HL) 5 6 6
(IX/Y+n) 7 8 8
