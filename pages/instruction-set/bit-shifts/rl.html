 RL op8
Arguments op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Rotate left
Description Rotates the contents of op8 left by one bit. The leaving (7th) bit is moved into the carry and the carry is moved into the first bit of op8.
Uses Reading bit-wise data, most significant bit first (left-to-right)
Byte Count 2
Clock Cycles
Operand RL RL.L RL.S
(HL) 5 6 6
(IX/Y+n) 7 8 8
op8 2 N/A N/A
