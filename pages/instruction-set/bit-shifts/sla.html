  SLA op8
Arguments op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Shift left arithmetic
Description Rotates the contents of op8 left by one bit. The leaving (7th) bit is moved into the carry, and the first bit of op8 is reset. (Quite different from SRA)
Uses Very quickly multiplying an 8-bit number by two; if the number becomes greater than 255, the carry flag will be set (else reset)
Byte Count 2
Clock Cycles
Operand SLA SLA.L SLA.S
(HL) 5 6 6
(IX/Y+n) 7 8 8
