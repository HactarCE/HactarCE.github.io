 RLC op8
Arguments op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Rotate left with carry
Description Rotates the contents of op8 left by one bit. The leaving (7th) bit is copied into the carry, and into the last bit of op8.
Uses Reading bit-wise data, most significant bit first (left-to-right), while preserving the original data (after it is performed eight times)
Byte Count 2
Clock Cycles
Operand RLC RLC.L RLC.S
(HL) 5 6 6
(IX/Y+n) 7 8 8
