 SRA op8
Arguments op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Shift right arithmetic
Description Rotates the contents of op8 right by one bit, but the 7th bit stays the same. The leaving (0th) bit is moved into the carry. (Quite different from SLA)
Uses Maybe helpful with signed integers? Potentially useful to set all bits of A to equal the 7th bit?
Byte Count 2
Clock Cycles
