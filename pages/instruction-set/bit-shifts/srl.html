 SRL R
Arguments r - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Shift right logical
Description Rotates the contents of r right by one bit. The leaving (0th) bit is moved into the carry, and the first bit of r is reset. (Similar to SLA)
Uses Very quickly dividing an 8-bit number by two. If the number was odd, the carry flag will be set (else reset).
Byte Count 2
Clock Cycles
