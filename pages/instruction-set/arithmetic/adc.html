ADC M,N Add with carry
Description Adds M, N, and the carry flag (+0 or +1) and stores the result in M.
M += N + Carry
Uses Storing the state of the carry flag in either A or HL with adc a,0 or ld bc/de,0 \ adc hl,bc/de. Can be useful otherwise in very specific situations.
Results
Register/Flag8-bit16-bit (non-ADL)24-bit (non-ADL)
M Set to result of M + N + Carry
S flag Set if result is negative; else reset
Z flag Set if result is 0; else reset
H flag (8-bit) Set if carry from bit 3; else reset
Set if carry from bit 11; else reset
P/V flag Set if overflow; else reset
N flag Reset
C flag Set if carry from bit 7; else reset Set if carry from bit 15; else reset Set if carry from bit 23; else reset
