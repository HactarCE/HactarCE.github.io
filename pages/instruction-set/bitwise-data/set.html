SET N,op8
Arguments N - A hard-coded integer from 0-7
op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Set the specific bit
Description Sets the Nth bit of op8
Uses Setting single bits. This can also be useful to set OS flags, for example 
SET	testInverse,(IY+textFlags)
Byte Count 2
Clock Cycles
Operand BIT BIT.L BIT.S
(HL) 3 4 4
(IX/Y+n) 5 6 6
r 2 N/A N/A
