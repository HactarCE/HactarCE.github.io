 BIT N,op8
Arguments N - A hard-coded integer from 0-7
op8 - Any 8-bit register or (HL),(IX+n),(IY+n)
Summary Test if the specific bit is set
Description Returns the opposite of the specified bit into the Z flag (ie. If the specified bit is zero, the Z flag is set; if the bit is not zero, the Z flag is reset)
Uses Testing single bits. This can also be useful to check OS flags, for example 
BIT	testInverse,(IY+textFlags)
Byte Count 2
Clock Cycles
Operand RL RL.L RL.S
(HL) 5 6 6
(IX/Y+n) 7 8 8
